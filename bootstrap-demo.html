How This Relates to Aurora Brews

The entire Aurora Brews website uses Bootstrap's grid system for layout, ensuring it looks great on mobile phones, tablets, and desktops. The navigation bar, buttons, cards displaying menu items, and this very alert box are all Bootstrap components, styled to match our cafe's brand.
